Consecutive frame pairs are subtracted via absolute differencing. Morphological opening removes single-pixel noise. A binary threshold τ produces a motion mask — only pixels that changed are lit.

Scoring Formula

Score = D × S × (1 + 0.1·L) × (1 + 0.05·V)
D = Net displacement (px) S = Straightness = D / Path L = Track length (frames) V = Avg speed (px/frame)

Algorithm Pseudocode

# Single-Object Asteroid Tracing Pipeline
Input:
FITS frames {I_t}
Output:
Best track T*
1: Load frames; detect shape-mismatch boundaries
2: For each valid pair (I_t, I_{t+1}):
3:   I8_t ← ZScaleNorm + GaussianBlur(I_t)
4:   M_t ← Threshold(Open(|I8_t − I8_{t+1}|))
5:   B_t ← BlobDetect(M_t, area=[4,500])
6: Build tracks via greedy NN linking (MAX_LINK_DIST)
7: Filter: length ≥ 4 AND displacement ≥ 8 px
8: Score: D × S × (1+0.1·L) × (1+0.05·V)
9: Return highest-scoring track T*

# Complexity: O(A_t · B_t) per transition

Paper Reference

Abstract
Detecting and tracing small moving objects in astronomical time-series is challenging due to low signal-to-noise ratio, background star fields, and acquisition discontinuities across sequences. This paper presents a lightweight single-object asteroid tracing pipeline for FITS time-series that combines ZScale-based intensity normalization, frame differencing for motion segmentation, blob extraction using area-constrained detection, and greedy multi-frame association for track formation. Candidate tracks are filtered and ranked using a motion-focused scoring function that favors sufficient displacement, linearity (straightness), and temporal continuity. Experiments were conducted on approximately 4000 FITS frames containing multiple sequences, where frame-size changes are treated as sequence boundaries to avoid invalid differencing. The system outputs an interpretable trajectory overlay and quantitative motion statistics (path length, displacement, and average pixel speed). In a representative run, the selected object was tracked for 18 frames with 96.1 px net displacement and 9.7 px/frame average speed. The results suggest that combining simple difference imaging with heuristic track scoring can produce stable tracklets under noisy backgrounds while remaining computationally inexpensive compared to heavier tracking frameworks.
